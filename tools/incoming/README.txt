Pipoya A안 — itch.io에서 무료 다운로드 후 이 폴더에 zip 넣고 npm run setup:pipoya

1) 몬스터 (필수)
   https://pipoya.itch.io/free-rpg-monster-pack
   → Pipoya RPG Monster Pack.zip

2) 캐릭터 (권장)
   https://pipoya.itch.io/pipoya-free-rpg-character-sprites-32x32
   → PIPOYA FREE RPG Character Sprites 32x32.zip

3) 던전 타일 (선택)
   https://pipoya.itch.io/pipoya-rpg-tileset-32x32
   → FREE RPG Tileset 32x32.zip

던전 배경 이미지는 public/assets/dungeon/dungeon_01.png ~ dungeon_18.png
(960x360 PNG) 로 직접 넣어도 됩니다.
