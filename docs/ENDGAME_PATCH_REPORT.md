# 투닥투닥RPG v4.0 엔드게임 패치 보고서

**버전:** 4.0.0  
**일자:** 2026-06-11  
**범위:** 싱글플레이 엔드 콘텐츠 (길드 레이드 제외)

---

## 1. 패치 목적

18지역(경기광주) 클리어 후에도 약 **1개월 이상** 플레이할 수 있는 오프라인 엔드 콘텐츠를 추가했습니다.  
멀티·길드·랭킹·로그인 없이 **로컬 세이브만**으로 진행합니다.

---

## 2. 해금 조건

| 조건 | 설명 |
|------|------|
| maxRegion ≥ 18 | 18번 지역까지 진행 |
| badges에 18 포함 | 경기광주 배지 획득 |
| boss_final 처치 | 최종 보스(파멸의 군주) 1회 이상 |

해금 후 하단 탭 **🌌 차원** 에서 이용합니다.

---

## 3. 신규 시스템

### 3.1 차원 균열 (50층)

- **일일 열쇠 6개** — 하루에 최대 6층 도전 (약 8~9일이면 50층, 여유 플레이 시 ~1개월)
- 3구역: 폐역(1~20) · 거울(21~40) · 심연(41~50)
- 속성 순환(화·수·뇌·독·무) — DPS 게이트 방식 (전투 연출 없이 즉시 판정)
- 보상: 골드, **공허 파편**, **균열 결정**(5·10층 단위)
- 5층마다 **유물** 1개 자동 획득

### 3.2 무한의 탑

- **일일 도전 12회**
- 주간 시드 기반 **변경 규칙** 6종 (화염/침수/뇌전/공허/거울/평온)
- 개인 최고 층 기록 · 주간 최고 층 별도 집계
- 10·25·50·75·100층 유물 마일스톤

### 3.3 유물 (18종)

- 균열 10종 + 탑 5종 + 도감 완성 + 각성 3명 + 전원 각성
- 영구 패시브: ATK/HP/DEF/치명/공속/EXP/골드 %
- `StatCalculator`·사냥·오프라인 보상에 반영

### 3.4 전설 각성 (캐릭터별)

- 조건: Lv.80+, **최종 전직** 완료, 균열 8층+, 비용 🪙250,000 · 공허×25 · 결정×3
- 효과: 해당 캐릭터 **전 스탯 +28%** (prestige 4 고정)
- 전원 각성 시 유물 **전설 모험단** (+10% 전 스탯)
